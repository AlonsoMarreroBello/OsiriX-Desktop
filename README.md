# OsiriX-Desktop

## How to start the project

To start the project, you must first install the npm dependencies:

```bash
npm install
```

Then, to start the project, run the following command:

```bash
npm start
```

This command will start the desktop application withing a development environment.

### Development tools

To view the developmet tools use the following command:

```bash
ctrl + shift + i
```
